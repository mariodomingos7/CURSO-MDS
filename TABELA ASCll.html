TABELA ASCII

A Tabela ASCII (American Standard Code for Information Interchange) é usada pela maior parte da indústria de computadores para a troca de informações. A tabela ASCII original possui 120 caracteres, aqueles com código entre 0 e 127.
(Veja na tabela abaixo a correspondência entre caractere e número).

Existem várias extensões da tabela ASCII utilizando também os números negativos de -1 a -128. Normalmente, essas tabelas estendidas incluem os caracteres acentuados.
